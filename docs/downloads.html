Notes
  • macOS installer packages install `Codex++ 管理工具.app` and `Codex++.app` into `/Applications`.
  • If macOS blocks the unsigned apps, run xattr -cr "/Applications/Codex++ 管理工具.app" and xattr -cr "/Applications/Codex++.app".
  • Windows x64 is for traditional Intel/AMD PCs. Windows arm64 is for Windows on ARM devices.
  • The manager still expects a local Codex installation and keeps the current Go refactor feature scope.
